# Future Features (Post-V1)

## Tier 1 — High Value, Medium Effort

- **Deadline reminders** via email/SMS when saved opportunities approach deadlines
- **Saved opportunities** with status tracking (discovered → applied → heard back)
- **LLM-powered cold email drafts** personalized to each opportunity + student profile
- **Resume gap analysis** per opportunity — specific suggestions for what to add/rewrite
- **Expanded data sources**: NSF REU API, USAJobs, SerpApi, PathwaysToScience

## Tier 2 — High Value, High Effort

- **Semantic matching** using sentence-transformers + pgvector for beyond-keyword relevance
- **Multi-university support** with config-driven scraper registry
- **User accounts** with persistent profiles and recommendation history
- **React + FastAPI production frontend** replacing Streamlit
- **Handshake EDU API integration** via UIUC career services partnership

## Tier 3 — Speculative / Long-Term

- **Reverse matching**: professors/labs post what they need, students get matched
- **Application tracker** with Kanban-style status board
- **Peer recommendations**: "Students like you also applied to..."
- **Professor/lab research interest database** built from publication analysis
- **International student CPT/OPT timeline advisor**
- **Resume bullet optimization** using LLM + opportunity requirements
- **Interview prep** tailored to research position expectations
- **Chrome extension** for one-click "analyze this opportunity" on any webpage
- **Startup/part-time opportunity aggregation** from local sources
- **Mentor feedback workflows**: advisors review student applications before submission
